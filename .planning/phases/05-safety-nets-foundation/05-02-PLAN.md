<task type="auto">
  <name>Task 2: Configure next/font/local and update typography</name>
  <files>src/app/layout.tsx, src/app/globals.css</files>
  <action>
**In src/app/layout.tsx:**

1. Remove the Inter import: `import { Inter } from 'next/font/google'` and the `inter` const
2. Add localFont import: `import localFont from 'next/font/local'`
3. Configure both fonts:

```typescript
const clashDisplay = localFont({
  src: './fonts/ClashDisplay-Variable.woff2',
  variable: '--font-clash-display',
  display: 'swap',
  weight: '200 700',
})

const generalSans = localFont({
  src: './fonts/GeneralSans-Variable.woff2',
  variable: '--font-general-sans',
  display: 'swap',
  weight: '200 700',
})
```

4. Update the `<html>` tag className to NOT include `inter.variable`. Instead just put the new font variables:
```html
<html lang="en" className={`${clashDisplay.variable} ${generalSans.variable}`}>
```

5. Update the `<body>` className: replace `${inter.variable} font-sans` with `font-sans` (font-sans will be overridden by CSS)

**In src/app/globals.css:**

Add these CSS variable definitions (at the top level, inside or after the existing `:root` block):

```css
:root {
  /* Typography fonts */
  --font-display: var(--font-clash-display), ui-sans-serif, system-ui, sans-serif;
  --font-body: var(--font-general-sans), ui-sans-serif, system-ui, sans-serif;

  /* Typography scale */
  --text-display-xl: 4.5rem;   /* 72px */
  --text-display: 3.75rem;     /* 60px */
  --text-heading-1: 3rem;      /* 48px */
  --text-heading-2: 2.25rem;   /* 36px */
  --text-heading-3: 1.875rem;  /* 30px */
  --text-heading-4: 1.5rem;    /* 24px */
  --text-body-lg: 1.125rem;    /* 18px */
  --text-body: 1rem;           /* 16px */
  --text-body-sm: 0.875rem;    /* 14px */
  --text-caption: 0.75rem;     /* 12px */
}

body {
  font-family: var(--font-body);
}

h1, h2, h3, h4, h5, h6 {
  font-family: var(--font-display);
}
```

If there is already a `body` rule with `font-family`, update it. If there are already heading rules with Inter or other fonts, update them.

Do NOT remove other existing CSS in globals.css — only add/modify font-related rules.

Also search for any Tailwind `font-sans` utility class configuration. If Tailwind is configured with a custom `fontFamily.sans`, update it to use `var(--font-body)`. Check `tailwind.config.ts` or `tailwind.config.js`.
