**Scoring Algorithm:**

Static factors (max 50 points):
- Source quality: referral=30, organic/seo=25, website=22, paid/ppc/lsa=20, facebook=18, manual=15, other=10
- Service value: hardscaping/commercial/patio/retaining wall=20, fertilization/aeration/mowing=12, cleanup/mulch/gutter/snow=8, other=5
- Contact completeness: phone=5, email=5, address=5

Engagement factors (max 30 points):
- Has been sent follow-up emails (any): +5
- Has been sent follow-up SMS (any): +5
- Follow-up stage >= 2 (engaged past initial contact): +10
- Follow-up stage >= 4 (deep engagement): +10 more

Recency factors (max 20 points):
- Created < 1 day ago: +20
- Created < 3 days ago: +15
- Created < 7 days ago: +10
- Created < 14 days ago: +5
- Created < 30 days ago: +2

Tier thresholds: hot >= 70, warm >= 40, cold < 40

**Workflow structure:**

5. **Calculate Scores** (code node, typeVersion 2)
   - ID: calculate-scores
   - Code:
     ```javascript
     const leads = $('Fetch Leads').all().map(i => i.json);
     const emailHistory = $('Fetch Email Send History').all().map(i => i.json);
     const smsHistory = $('Fetch SMS Send History').all().map(i => i.json);

     // Build engagement lookup sets
     const emailedLeads = new Set(emailHistory.map(e => (e.contact_email || '').toLowerCase()));
     const smsedLeads = new Set(smsHistory.map(s => (s.to_phone || '').replace(/[^\d+]/g, '')));

     const results = [];

     for (const lead of leads) {
       // Skip converted leads
       if (lead.status === 'converted' || lead.status === 'lost') continue;

       let score = 0;
       const factors = [];

       // --- Static factors (max 50) ---

       // Source quality
       const source = (lead.source || '').toLowerCase();
       const sourceScores = { referral: 30, organic: 25, seo: 25, website: 22, paid: 20, ppc: 20, lsa: 20, google_ads: 20, facebook_ads: 18, facebook: 18, manual: 15, phone: 20 };
       const sourceScore = sourceScores[source] || 10;
       score += sourceScore;
       factors.push(`Source: ${source} (+${sourceScore})`);

       // Service value
       const service = (lead.service || '').toLowerCase();
       const highValue = ['hardscaping', 'commercial', 'patio', 'retaining wall', 'landscape design', 'irrigation', 'outdoor kitchen'];
       const midValue = ['fertilization', 'aeration', 'overseeding', 'mowing', 'lawn care', 'lawn maintenance', 'garden bed', 'mulching', 'bush trimming'];
       const cleanupValue = ['cleanup', 'leaf', 'gutter', 'snow', 'spring cleanup', 'fall cleanup'];
       let serviceScore = 5;
       if (highValue.some(s => service.includes(s))) { serviceScore = 20; }
       else if (midValue.some(s => service.includes(s))) { serviceScore = 12; }
       else if (cleanupValue.some(s => service.includes(s))) { serviceScore = 8; }
       score += serviceScore;
       factors.push(`Service: ${service || 'unknown'} (+${serviceScore})`);

       // Contact completeness
       if (lead.phone) { score += 5; factors.push('Has phone (+5)'); }
       if (lead.email) { score += 5; factors.push('Has email (+5)'); }
       if (lead.address) { score += 5; factors.push('Has address (+5)'); }

       // --- Engagement factors (max 30) ---
       const email = (lead.email || '').toLowerCase();
       const phone = (lead.phone || '').replace(/[^\d+]/g, '');
       const followUpStage = lead.follow_up_stage || 0;

       if (email && emailedLeads.has(email)) { score += 5; factors.push('Received follow-up email (+5)'); }
       if (phone && smsedLeads.has(phone)) { score += 5; factors.push('Received follow-up SMS (+5)'); }
       if (followUpStage >= 2) { score += 10; factors.push(`Follow-up stage ${followUpStage} >= 2 (+10)`); }
       if (followUpStage >= 4) { score += 10; factors.push(`Follow-up stage ${followUpStage} >= 4 (+10)`); }

       // --- Recency factors (max 20) ---
       const daysOld = (Date.now() - new Date(lead.created_at).getTime()) / (1000 * 60 * 60 * 24);
       let recencyScore = 0;
       if (daysOld < 1) recencyScore = 20;
       else if (daysOld < 3) recencyScore = 15;
       else if (daysOld < 7) recencyScore = 10;
       else if (daysOld < 14) recencyScore = 5;
       else if (daysOld < 30) recencyScore = 2;
       score += recencyScore;
       factors.push(`Recency: ${Math.round(daysOld)}d (+${recencyScore})`);

       // Final score
       const finalScore = Math.min(score, 100);
       const newTier = finalScore >= 70 ? 'hot' : finalScore >= 40 ? 'warm' : 'cold';
       const wasHot = lead.lead_tier === 'hot';
       const nowHot = newTier === 'hot' && !wasHot; // Newly became hot

       results.push({
         json: {
           lead_id: lead.id,
           first_name: lead.first_name,
           last_name: lead.last_name,
           phone: lead.phone,
           service: lead.service,
           lead_score: finalScore,
           lead_tier: newTier,
           previously_hot: wasHot,
           newly_hot: nowHot,
           score_factors: factors
         }
       });
     }

     return results;
     ```
