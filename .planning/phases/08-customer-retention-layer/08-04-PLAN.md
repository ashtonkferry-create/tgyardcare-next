# Plan 08-04: Service Actions + Referral Engine

## Goal
Request additional service modal, crew rating modal with star animation, referral card with copy link, and the public /r/[code] personalized landing page.

## Files to Create

### src/components/portal/RequestServiceModal.tsx (client)
Modal triggered by "Request a Service" button.
- Service type dropdown (15 services from existing STATIC_SERVICES list)
- Preferred date picker (input type="date")
- Notes textarea
- Submits POST to /api/portal/request-service
- Success: animated checkmark, modal auto-closes
- Motion: AnimatePresence slide-up

### src/app/api/portal/request-service/route.ts
POST handler:
- Authenticates user via createClient().auth.getUser()
- Gets portal_customer by user_id
- Inserts into service_requests table
- Returns 200 on success

### src/components/portal/RateCrewModal.tsx (client)
Star rating modal triggered from service history "Rate this visit".
- Props: serviceRecordId
- 5 stars with hover animation (each star grows + glows on hover)
- Optional comment textarea
- Submits POST to /api/portal/rate-service
- On submit: animated gold stars fill, "Thank you!" message
- Uses AnimatePresence

### src/app/api/portal/rate-service/route.ts
POST handler:
- Authenticates user
- Validates rating 1-5
- Inserts into service_ratings (upsert — one rating per service_record)
- Returns 200

### src/components/portal/ReferralCard.tsx (client)
Shows referral link + stats in dashboard.
- Customer's unique link: tgyardcare.com/r/{code}
- "Copy Link" button with clipboard animation (button text changes to "Copied! ✓" for 2s)
- Stats: referrals sent, pending, qualified, credited
- Credit display: "$X earned" in green
- How it works: 3 step mini-timeline (Share → They Book → You Both Get $50)
- Motion: border-beam on card edge
