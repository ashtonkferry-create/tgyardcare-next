# Plan 08-03: Portal Dashboard — God Tier UI

## Goal
Build the authenticated customer dashboard. Dark-themed, premium, animated. Matches TotalGuard brand. Pulls data from portal_customers, service_records, upcoming_jobs, customer_invoices.

## Architecture
Server Component page.tsx → fetches user session + portal_customer + all data → passes to client islands for animation.

## Files to Create

Layout for all /portal/* pages (excluding login/auth).
- Checks auth session server-side, redirects to /portal/login if not authed
- Renders PortalNav (sidebar desktop, bottom bar mobile)
- Dark background #050d07

### src/components/portal/PortalNav.tsx (client)
Navigation sidebar (desktop) / bottom tab bar (mobile).
- Tabs: Dashboard, Schedule, History, Invoices, Referral
- Active state: green (#22c55e) underline/highlight
- Sign out button → supabase.auth.signOut() → router.push('/portal/login')
- Logo: "TotalGuard Portal" with green spark

### src/app/portal/dashboard/page.tsx (server component)
Fetches all data server-side:
```typescript
const supabase = await createClient()
const { data: { user } } = await supabase.auth.getUser()
// redirect if no user
const { data: customer } = await supabase.from('portal_customers').select('*').eq('user_id', user.id).single()
// if no customer, create one with auto-generated referral_code
const { data: upcoming } = await supabase.from('upcoming_jobs').select('*').eq('customer_id', customer.id).gte('scheduled_date', today).order('scheduled_date').limit(3)
const { data: history } = await supabase.from('service_records').select('*').eq('customer_id', customer.id).order('service_date', { ascending: false }).limit(5)
const { data: invoices } = await supabase.from('customer_invoices').select('*').eq('customer_id', customer.id).order('invoice_date', { ascending: false }).limit(5)
const { data: referrals } = await supabase.from('referral_events').select('*').eq('referral_code', customer.referral_code)
```

Layout: responsive CSS grid
- Mobile: stacked cards
- Desktop: 2-col grid (upcoming + referral left, history + invoices right)
