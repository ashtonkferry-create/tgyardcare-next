# Plan 10-03: Internal Link Web + Breadcrumbs

## Goal
Build visible Breadcrumb component, add "Serving Across Dane County" section to all 8 service pages, and create InlineServiceLinks for blog auto-linking.

## Part A: Breadcrumb Component
`src/components/Breadcrumb.tsx` (server component, no 'use client')

```typescript
interface BreadcrumbItem { label: string; href?: string; }
interface BreadcrumbProps { items: BreadcrumbItem[]; }
```

Design:
- Small text (text-xs / text-sm), muted color (rgba(255,255,255,0.45))
- Separator: `/` or `›` in even more muted color
- Current page: white, no link
- Hover: green color on links
- JSON-LD BreadcrumbList schema embedded via <script type="application/ld+json">

Usage on city-service pages: `Home › Service Areas › {City} › {Service}`
Usage on service pages: `Home › Services › {Service}`
Usage on blog posts: `Home › Blog › {Category} › {Title}`
Usage on location pages: `Home › Service Areas › {City}`

## Part B: "Serving Across Dane County" on All 8 Service Pages
Add a new section to each of the 8 service ContentFiles.

Section design:
- Headline: "We Provide {Service} Across Dane County"
- Subheadline: "Same quality. Same team. Every city we serve."
- 12-city grid (3-col desktop, 2-col mobile):
  Each city card: City name + "→" arrow, links to /{service}-{city}-wi
- Green-accented cards with hover effect (translateY -2px)
- Full-width section with dark background

Service slug mapping:
- MowingContent → lawn-mowing
- FertilizationContent → fertilization-weed-control
- GutterCleaningContent → gutter-cleaning
- GutterGuardsContent → gutter-guard-installation
- FallCleanupContent → fall-cleanup
- SpringCleanupContent → spring-cleanup
- SnowRemovalContent → snow-removal
- HardscapingContent → hardscaping

Insert section before the Footer in each file.

## Part C: InlineServiceLinks Component
`src/components/InlineServiceLinks.tsx` (client component)

Wraps blog content HTML and adds contextual links to service/city keywords.

```typescript
const KEYWORD_MAP: Record<string, string> = {
  'lawn mowing': '/services/mowing',
  'lawn care': '/services/mowing',
  'gutter cleaning': '/services/gutter-cleaning',
  'gutter guards': '/services/gutter-guards',
  'snow removal': '/services/snow-removal',
  'fertilization': '/services/fertilization',
  'fall cleanup': '/services/fall-cleanup',
  'spring cleanup': '/services/spring-cleanup',
  'hardscaping': '/services/hardscaping',
  'Madison lawn': '/lawn-mowing-madison-wi',
  'Middleton lawn': '/lawn-mowing-middleton-wi',
  // ... 12 city variants for top 3 services
};

// Max 3 links per keyword instance to avoid over-linking
```

Props: `{ content: string }` — renders as dangerouslySetInnerHTML but with links injected.
