# Plan 10-02: Dynamic City-Service Route (96 Pages)

## Goal
Create the dynamic route that generates all 96 city-service pages at build time with generateStaticParams + unique content per page.

## File: src/app/[cityService]/page.tsx

This is a Server Component. NO 'use client'.

### Key constraints
- Must not conflict with existing static routes. The `[cityService]` segment ONLY matches the `{service}-{city}-wi` pattern. Add a check at the top: if the param doesn't match the pattern, call `notFound()`.
- Existing static pages (`/lawn-care-madison-wi`, `/gutter-cleaning-madison-wi`, `/snow-removal-madison-wi`) will be superseded by this route. Add 301 redirect entries in Supabase seo_redirects for old URLs.

### generateMetadata

```typescript
export async function generateMetadata({ params }) {
  const content = parseCityService(params.cityService);
  if (!content) return {};
  return {
    title: content.title,
    description: content.description,
    alternates: { canonical: content.canonical },
    openGraph: { title: content.title, description: content.description },
  };
}
```
