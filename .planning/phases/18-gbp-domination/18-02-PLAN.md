     - Photo count (15 pts): 20+ = 15, 10-19 = 10, 5-9 = 5, <5 = 0
     - FAQ freshness (15 pts): updated within 90 days = 15, 180 days = 10, older = 0
     - Hours accuracy (10 pts): hours present = 10, missing = 0
   - Grade: A (90+), B (75-89), C (60-74), D (40-59), F (<40)
   - Compare to previous month score if available

5. Store Score (id: tg136-store, name: "Store Optimization Score")
   - type: n8n-nodes-base.httpRequest, typeVersion: 4.2
   - POST https://lwtmvzhwekgdxkaisfra.supabase.co/rest/v1/gbp_scores

6. Send Report (id: tg136-report, name: "Send GBP Score Report to Telegram")
   - type: n8n-nodes-base.telegram, typeVersion: 1.2
