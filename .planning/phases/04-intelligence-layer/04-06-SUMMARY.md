# Phase 4 Plan 6: TG-118 Weekly Intelligence Assembler Summary

Shared HTML email assembly sub-workflow accepting structured report sections and returning branded, mobile-friendly HTML with plain text fallback.

## What Was Built

### TG-118 Weekly Intelligence Assembler
- **n8n ID:** rOeTPPi2kW6thURB (ACTIVE)
- **Type:** Sub-workflow (executeWorkflowTrigger typeVersion 1)
- **Nodes:** Execute Workflow Trigger -> Build HTML Email -> Return HTML

**Input contract:**
```json
{
  "report_title": "Weekly Revenue Report",
  "report_subtitle": "March 10-16, 2026",
  "sections": [
    { "type": "metrics", "title": "Key Metrics", "items": [{ "label": "Revenue", "value": "$4,200", "change": "+12%" }] },
    { "type": "highlights", "title": "Highlights", "items": ["Record week for new customers"] },
    { "type": "table", "title": "Top Services", "headers": ["Service", "Revenue"], "rows": [["Mowing", "$1,800"]] },
    { "type": "anomalies", "items": [{ "severity": "warning", "message": "Ad spend 30% above normal" }] },
    { "type": "actions", "items": ["Review ad budget allocation"] }
  ]
}
```

**Output:** `{ html, plain_text }`

**HTML features:**
- 600px max-width, table-based layout (email-safe)
- All CSS inline (no style blocks)
- Dark green (#1a5d1a) header bar with TotalGuard branding
- Metric cards with large numbers + green/red change badges
- Striped data tables with right-aligned numbers
- Green left-border highlight bullets
- Orange/red left-border anomaly alerts with severity badges
- Numbered action items with green circle indicators
- Footer: "Generated by TotalGuard Intelligence Layer" + timestamp
- Empty sections skipped gracefully
