  <action>
Create a new n8n workflow JSON for TG-127-pre-job-notification. This is a sub-workflow called by TG-05 via executeWorkflow when a job_scheduled email is parsed.

**Workflow flow:**

1. **Execute Workflow Trigger** (id: tg127-trigger, name: "When Called by TG-05")
   - type: n8n-nodes-base.executeWorkflowTrigger, typeVersion: 1
   - Receives: customer_name, customer_phone, customer_email, address, service, scheduled_date, scheduled_time, event_id

2. **Calculate Wait Duration** (id: tg127-calc-wait, name: "Calculate Wait Until Day Before")
   - type: n8n-nodes-base.code, typeVersion: 2
   - jsCode logic:
     ```
     const input = $input.first().json;
     const scheduledDate = input.scheduled_date; // YYYY-MM-DD or null
     const now = new Date();

     if (!scheduledDate) {
       // No date parsed -- skip pre-job notification
       return [{ json: { ...input, should_send: false, reason: 'no_scheduled_date' } }];
     }

     const jobDate = new Date(scheduledDate + 'T09:00:00');
     const dayBefore = new Date(jobDate);
     dayBefore.setDate(dayBefore.getDate() - 1);
     dayBefore.setHours(16, 0, 0, 0); // Send at 4 PM day before

     if (dayBefore <= now) {
       // Job is tomorrow or today -- send immediately
       return [{ json: { ...input, should_send: true, send_now: true } }];
     }

     // Calculate wait in minutes
     const waitMinutes = Math.floor((dayBefore.getTime() - now.getTime()) / 60000);
     return [{ json: { ...input, should_send: true, send_now: false, wait_minutes: waitMinutes } }];
     ```

3. **Should Send?** (id: tg127-if-send, name: "Should Send Pre-Job SMS?")
   - type: n8n-nodes-base.if, typeVersion: 2
   - Condition: $json.should_send equals true
   - True branch -> continues
   - False branch -> ends (no notification if no date)

4. **Should Wait?** (id: tg127-if-wait, name: "Need to Wait?")
   - type: n8n-nodes-base.if, typeVersion: 2
   - Condition: $json.send_now equals false
   - True (needs wait) -> Wait node
   - False (send now) -> Send SMS directly

5. **Wait Until Day Before** (id: tg127-wait, name: "Wait Until Day Before Service")
   - type: n8n-nodes-base.wait, typeVersion: 1.1
   - amount: "={{ $json.wait_minutes }}"
   - unit: "minutes"

6. **Build SMS Payload** (id: tg127-build-sms, name: "Build Pre-Job SMS")
   - type: n8n-nodes-base.code, typeVersion: 2
   - jsCode: Build the SMS message. Include service type and time if available.
     ```
     const input = $input.first().json;
     const name = input.customer_name || 'there';
     const service = input.service ? ` for ${input.service}` : '';
     const time = input.scheduled_time ? ` around ${input.scheduled_time}` : '';

     const message = `Hi ${name}! This is TotalGuard Yard Care. Just a friendly reminder that we'll be at your property tomorrow${service}${time}. Please make sure gates are unlocked and pets are secured. Questions? Call 608-535-6057`;

     return [{
       json: {
         to_phone: input.customer_phone,
         message_body: message,
         workflow_name: 'TG-127-pre-job-notification',
         message_type: 'pre_job_reminder'
       }
     }];
     ```

7. **Send via TG-94** (id: tg127-send-sms, name: "Send Pre-Job SMS via TG-94")
   - type: n8n-nodes-base.executeWorkflow, typeVersion: 1
   - workflowId: "AprqI2DgQA8lehij" (TG-94 unified SMS sender)

**Connections:**
- tg127-trigger -> tg127-calc-wait -> tg127-if-send
- tg127-if-send true -> tg127-if-wait
- tg127-if-wait true (needs wait) -> tg127-wait -> tg127-build-sms -> tg127-send-sms
- tg127-if-wait false (send now) -> tg127-build-sms -> tg127-send-sms

**Settings:** executionOrder: "v1", callerPolicy: "workflowsFromSameOwner"

**Deploy to n8n:** POST the workflow to `https://tgyardcare.app.n8n.cloud/api/v1/workflows` with the n8n API key. Then ACTIVATE it: POST to `/api/v1/workflows/{id}/activate`. Record the returned workflow ID.

Save the local JSON file with the returned n8n ID in the top-level `"id"` field.

7. **Analyze Sentiment** (id: tg128-sentiment, name: "Analyze Reply Sentiment")
   - type: n8n-nodes-base.code, typeVersion: 2
   - jsCode:
     ```
     const reply = ($input.first().json.Body || $input.first().json.body || $input.first().json.message || '').toLowerCase().trim();
     const from = $input.first().json.From || $input.first().json.from || $input.first().json.phone || '';

     // Positive indicators
     const positiveWords = ['great', 'good', 'awesome', 'perfect', 'love', 'amazing', 'excellent', 'wonderful', 'fantastic', 'thank', 'thanks', 'happy', 'beautiful', 'nice', 'yes', 'looks great', 'well done'];
     // Negative indicators
     const negativeWords = ['bad', 'terrible', 'awful', 'horrible', 'missed', 'wrong', 'not good', 'unhappy', 'disappointed', 'poor', 'complaint', 'problem', 'issue', 'no', 'nope', 'fix', 'redo', 'again', 'unacceptable'];

     const isPositive = positiveWords.some(w => reply.includes(w));
     const isNegative = negativeWords.some(w => reply.includes(w));

     let sentiment = 'neutral';
     if (isPositive && !isNegative) sentiment = 'positive';
     else if (isNegative) sentiment = 'negative';

     return [{ json: { sentiment, reply_text: reply, from_phone: from } }];
     ```
