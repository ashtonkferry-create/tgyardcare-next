# Plan 09-01: Blog Category System + Madison Lawn Care Guide Hub

## Goal
Add a category migration, rebuild the blog index with animated category filters, and create the authoritative /lawn-care-guide hub page.

## Part A: Blog Category Migration
```sql
-- Add category column if it doesn't exist (it may already exist)
ALTER TABLE blog_posts ADD COLUMN IF NOT EXISTS category text DEFAULT 'lawn-care';
ALTER TABLE blog_posts ADD COLUMN IF NOT EXISTS tags text[] DEFAULT '{}';

-- Seed categories on existing posts based on slug patterns
UPDATE blog_posts SET category = 'seasonal-tips' WHERE slug LIKE '%spring%' OR slug LIKE '%fall%' OR slug LIKE '%winter%' OR slug LIKE '%summer%';
UPDATE blog_posts SET category = 'service-guides' WHERE slug LIKE '%gutter%' OR slug LIKE '%mowing%' OR slug LIKE '%mulch%' OR slug LIKE '%fertiliz%';
UPDATE blog_posts SET category = 'how-to' WHERE slug LIKE '%how%' OR slug LIKE '%guide%' OR slug LIKE '%checklist%';
UPDATE blog_posts SET category = 'local-guides' WHERE slug LIKE '%madison%' OR slug LIKE '%dane%' OR slug LIKE '%wisconsin%';
-- Default remaining to lawn-care
UPDATE blog_posts SET category = 'lawn-care' WHERE category IS NULL;
```
